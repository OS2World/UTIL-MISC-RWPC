RWPC - Find duplicate files along the PATH
==========================================

1. Installation

rwpc consists of just one EXE, rwpc.exe. Copy it into some directory 
in your PATH.

2. Running rwpc

At a command prompt, enter

       rwpc [-e <ENV>] [<filespec>]

By default, rwpc will look for *.exe along PATH.

To look for duplicate help files, you might enter:

       rwpc -e HELP *.hlp
